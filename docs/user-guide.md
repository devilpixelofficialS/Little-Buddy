# User Guide

## Getting Started

### First Launch

1. Open Little Buddy
2. Register a new account or login
3. Start chatting with your assistant

### Main Interface

The app has a sidebar for navigation and a main chat area with a voice orb at the bottom.

## Chat Features

### Sending Messages

1. Type in the input field at the bottom
2. Press Enter or click Send
3. Assistant responds in the chat area

### Voice Input

1. Click the Voice Orb (center bottom)
2. Speak when the orb pulses
3. Your speech is transcribed automatically
4. Assistant processes and responds

### Voice Orb States

| State | Meaning |
|-------|---------|
| Idle | Ready for input |
| Listening | Recording audio |
| Thinking | Processing request |
| Speaking | Playing response |
| Error | Error occurred |

## Navigation

| Page | Description |
|------|-------------|
| Assistant | Main chat interface |
| Memory | View and manage memories |
| Skills | Installed automation skills |
| Workflows | Create and run workflows |
| Settings | Configure the application |

## Memory System

Memories are created automatically from conversations. View them on the Memory page with search functionality.

## Skills

Built-in skills include browser, desktop, filesystem, memory, vision, and workflow automation. Skills are invoked automatically or can be run manually from the Skills page.

## Workflows

Create automation workflows with steps like agent tasks, skill calls, conditions, delays, and transforms. Run them manually or set up triggers.

## Settings

Configure voice, memory, AI, and skills settings from the Settings page.
